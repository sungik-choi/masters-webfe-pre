# Breakpoint 란?
***중단점***이라고도 한다. 디버깅 시 특정 라인을 breakpoint로 지정하면, 해당 라인 직전까지 코드를 실행한다.

# Watch 사용법
***조사식***이라고도 한다. 디버깅 시 여러 변수들을 한번에 확인할 수 있다.
